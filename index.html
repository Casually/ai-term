小而美的现代终端利器

不改变你原有的终端习惯，只在需要时为你提供强大的 AI 辅助。

自然语言转命令

无需查阅 man 手册。用日常语言描述需求，AI 即刻输出精确的 shell 指令并准备就绪。

Autonomous Agent

不仅是生成器。AI 会读取命令执行的 stdout/stderr 结果，自我纠错直到任务彻底完成。

敏感操作拦截

系统级安全感。所有涉及文件修改、系统配置的敏感指令，都会强制暂停并等待您的 [y/N] 确认。

SSH 无缝联控

支持内置 /ssh 指令。随时将 AI 的思考与执行战场转移至您的远程服务器，轻松实现 AI 运维。

极致 Rust 性能

由 Rust 强力驱动，毫秒级冷启动，极低内存占用。编译为单一二进制文件，零环境依赖，即拷即用。

灵活接入模型

开放式设计，支持自定义配置。原生兼容 OpenAI 及 Google Gemini 等主流 LLM，随时切换背后的大脑。

三步开启 AI 终端之旅

极其简单的安装与配置，让你在 2 分钟内获得超能力。

1

下载或编译安装

您可以直接从 GitHub Release 页面下载适用于 macOS 或 Linux 的二进制文件，或者通过源码自行编译：

# 克隆仓库
git clone https://github.com/Casually/ai-term.git
cd ai-term
# 编译并运行 cargo build --release
./target/release/ai-term
2

配置 API Key

首次运行会在您的用户目录下生成 ~/.ai-term/config.toml 文件。使用任何文本编辑器打开它，填入您的模型配置：

[llm]
provider = "openai" # 或 gemini, 自定义
api_key = "sk-您的API密钥"
model = "gpt-4o"
3

开始对话

在 ai-term 终端中，直接用人类语言输入您的需求，然后按下 Shift + Enter (或 Ctrl + G) 召唤 AI Agent：

🤖 ai-term ❯ 帮我把当前目录所有 png 图片转换为 webp 格式，并删除原文件
[Shift+Enter] 触发 AI 思考与执行...

丰小子 OpenRouter 聚合 API

还在为注册海外平台、绑定信用卡而烦恼？我们强烈推荐使用国内领先的聚合 API 服务。一个 Key，极速直连全球最强的大语言模型（如 Claude 3.5 Sonnet, GPT-4o）。

如何配置使用

注册并获取您的专属 Token，修改您的 config.toml 即可无缝享受极速的 AI 编码体验。 

[llm]
provider = "openai" # 兼容 OpenAI 格式
api_key = "sk-xxxxxx" # 填入丰小子的 Token
base_url = "https://openrouter.fengxiaozi.net/api"
model = "dashscope/deepseek-v4-pro"
